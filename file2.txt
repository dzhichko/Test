22. В командной строке переместитесь в скачанный каталог Test с помощью 
команды 
cd Test
23. Просмотрите статус каталога. Если есть неотслеживаемые файлы, 
проиндексируйте их.
24. Выполните commit.
Если выполнить commit без комментария, то откроется окно, изображенное 
на рисунке ниже:
Для того чтобы закрыть его нажмите «esc», затем напишите «:wq» и нажмите 
«enter». Затем выполните commit с комментарием (параметром -m)
